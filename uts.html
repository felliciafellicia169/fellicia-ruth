Cara Kerja Model Machine Learning

Sistem menggunakan algoritma K-Nearest Neighbors (KNN) untuk memberikan rekomendasi berdasarkan kemiripan profil pengguna dengan data pelatihan yang tersedia.

1. Input Data

Pengguna mengisi data profil dan preferensi makanan

2. Preprocessing

Data diproses dan dinormalisasi untuk analisis

3. KNN Algorithm

Mencari pengguna dengan profil serupa dalam dataset

4. Rekomendasi

Menghasilkan rekomendasi berdasarkan pola yang ditemukan

Parameter Utama Model KNN:

  • n_neighbors: Jumlah tetangga terdekat yang digunakan (default: 5)
  • weights: Pembobotan berdasarkan jarak ('uniform' atau 'distance')
  • metric: Metrik jarak yang digunakan (Euclidean, Manhattan, dll.)
